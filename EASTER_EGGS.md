# WIHY App Easter Eggs / Demo Shortcuts

These are special search queries you can type into the home screen search bar to quickly navigate to demo screens with pre-populated mock data. Useful for testing and development.

## Search Shortcuts

| Search Query | Screen | Description |
|--------------|--------|-------------|
| `nutrition apple` | NutritionFacts | Demo apple nutrition data with health score, macros, vitamins |
| `beauty rose` | BeautyFacts | Demo rose oil serum with ingredient analysis |
| `pet chicken` | PetFoodFacts | Demo chicken dog food with pet nutrition info |
| `photo salad` | FoodPhotoFacts | Demo food photo analysis of a salad |
| `pill aspirin` | PillIdentification | Demo aspirin pill identification |
| `label organic` | LabelReader | Demo organic milk label with greenwashing analysis |

## Screen Details

### `nutrition apple` → NutritionFacts
- **Theme:** Blue (#3b82f6)
- **Data:** Fresh apple nutrition facts
- **Features:** Health score (95), Grade A, NOVA 1, macros, vitamins

### `beauty rose` → BeautyFacts
- **Theme:** Pink (#ec4899)
- **Data:** Rose Oil Face Serum by Beauty Botanica
- **Features:** Ingredient concerns, fragrance warning, cruelty-free

### `pet chicken` → PetFoodFacts
- **Theme:** Purple (#8b5cf6)
- **Data:** Chicken & Rice Adult Dog Food
- **Features:** Pet nutrition grade, protein sources, AAFCO approved

### `photo salad` → FoodPhotoFacts (NEW)
- **Theme:** Orange (#f97316)
- **Data:** Mixed Green Salad photo analysis
- **Features:** Detected foods, health score (92), nutrition facts, meal type

### `pill aspirin` → PillIdentification (NEW)
- **Theme:** Amber (#f59e0b)
- **Data:** Bayer Aspirin 325mg
- **Features:** Pill ID with imprint, shape, color, multiple matches

### `label organic` → LabelReader (NEW)
- **Theme:** Red (#ef4444)
- **Data:** Organic Valley Whole Milk
- **Features:** Greenwashing score, claim verification, certifications

## Usage

1. Open the WIHY app
2. Go to the Home screen
3. Tap the search bar
4. Type one of the exact shortcuts above (case-insensitive)
5. Press Enter/Search

## Notes

- Queries are case-insensitive (`Nutrition Apple` works the same as `nutrition apple`)
- Queries must match exactly (no partial matches)
- Any other search query goes to the AI chat
